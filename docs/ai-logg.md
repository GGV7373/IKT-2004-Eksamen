# AI-logg

## 2026-06-01

Det ble satt opp et enkelt Node.js- og Express-prosjekt for nettsiden.

Ferdig arbeid:
- Installerte `express` i prosjektet.
- Opprettet en server med oppstart i `index.js`.
- Laget en enkel SSR-løsning med Express.
- Opprettet `src/app.js` for Express-oppsettet.
- Opprettet `src/routes/site.js` for rotruten `/`.
- Laget en enkel nettside med overskrift, tekst og en knapp.
- La inn JavaScript i `public/js/main.js` slik at knappen viser teksten `Hei` når brukeren trykker.
- Verifiserte at serveren starter med `npm start` og kjører på `http://localhost:3000`.

Mappestruktur som ble brukt:
- `index.js` starter serveren.
- `src/` inneholder app og routes.
- `public/` inneholder filer som sendes til nettleseren.
